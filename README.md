# Rhythm Reserve

**Rhythm Reserve** is an Object-Oriented Programming (OOP) project designed to manage reservations for music practice rooms. The system helps users book, manage, and organize practice room schedules efficiently.

## Project Structure
The project follows Object-Oriented Programming concepts such as:
- Classes and objects
- Encapsulation
- Modular design for reservation management

## Diagram
